# FileSearch App

**Desktop search engine for Mac and Windows**

A desktop application for recursively searching file contents inside a selected folder.  
Alpha version available now in the [Releases](https://github.com/123Matthias/FileSearch/releases) area.

## Features

- Recursive file content search inside the selected folder
- English & German UI (switch in app options)
- Dark theme / Light theme switching with system mode
- GOTO App QHelp QOptions for detailed Information

## Installation (macOS)

1. Download the latest **.dmg** file from **Releases**.
2. Drag **FileSearch.app** into your **Applications** folder.
3. On first launch, an **Apple Security Alert** will appear. Click **OK**, **not Delete**.
4. Go to **System Settings** → **Privacy & Security**.
5. Scroll down and click **"Open Anyway"**.
6. Done. The app will now open normally from your **Applications** folder.

## Installation (Windows)

1. Download the latest **.exe** file from **Releases**.
2. Double-click the **.exe** file after downloading.
3. If the **"Unknown Developer"** message appears, click **More information**.
4. Then click **Install anyway**.

## Settings Folder Uninstalling

A folder named **`keysearch_app_settings`** will be created in your **home directory** on both **Windows** and **macOS**.  
You can remove this folder manually after uninstalling the app.
